# ChatGPT Projects 使用

1. 新建 ChatGPT Project，例如“ToMemo 内容生成器”。
2. 上传本目录全部文件。
3. 将 `00_Project_Instructions.txt` 全文复制到 Project Instructions。
4. 描述所需分类、标题、正文数量和语气。
5. 将返回 JSON 粘贴到网页的“粘贴 JSON”入口；网页会自动识别为 AI 内容包并进入预览。也可以先建立工作区，再使用专门的“AI 内容包”入口。
6. 在网页继续人工调整、校验并导出最终 ToMemo 配置。

AI 内容包不能直接导入手机，必须先由网页工具合并进真实配置。
