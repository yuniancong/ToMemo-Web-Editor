# ToMemo Web Editor v1.0.0

首个公开稳定版本，建立从 ToMemo 手机配置到桌面可视化编辑、AI 内容生成与手机回导的完整基础工作流。

## 核心能力

### 真实 ToMemo 配置工作台

- 导入 ToMemo `1.0` JSON，显示分类、Memo 列表和内容详情。
- 从空白工作区创建全新的分类与 Memo，无需已有手机配置。
- 编辑标题、正文与所属分类，生成符合真实导出格式的大写 UUID 和 UTC 时间。
- 导出前校验颜色、ID、时间、重复标识和分类引用。
- 保留当前编辑器不认识的字段，降低未来版本数据丢失风险。

### AI 内容包桥接

- 定义独立的 `tomemo-content-package` 格式，使 ChatGPT 等 AI 只生成标题与正文。
- 网页负责生成 ToMemo 内部 ID、时间和分类关系，避免 AI 直接修改底层配置。
- 支持 JSON 文件、直接粘贴以及 Markdown JSON 代码块。
- 提供导入预览、目标分类选择和完全重复项跳过。

### 桌面编辑体验

- macOS 风格三栏布局。
- 分类和 Memo 新增、编辑、移动、复制与删除。
- Command/Shift 多选和批量操作。
- 按标题、创建时间、更新时间和原始顺序查看。
- 插入已验证的 `{{CLIPBOARD}}`、`{{CURSOR}}` 动态变量。
- 撤销、重做和浏览器本地自动保存。

### 本地优先与离线支持

- 无服务器、账号、遥测、远程数据库或内置 AI API。
- 所有配置处理都在浏览器本地完成。
- 提供可安装的 PWA 和静态生产构建。
- 附带 ChatGPT Projects 知识库和手机验收配置。

## ToMemo 兼容性验证

- 基于两份真实 ToMemo `1.0` 导出文件完成 codec 验证。
- 真实手机导入成功，新 ID 作为新数据添加，既有内容未受干扰。
- ToMemo 导入界面确认相同 ID 会覆盖已有记录。

## 安装

```bash
git clone https://github.com/yuniancong/ToMemo-Web-Editor.git
cd ToMemo-Web-Editor
npm install
npm run dev
```

## 已知限制

- 仅确认支持 ToMemo 配置版本 `1.0`。
- 批量内容在手机端的时间排序一致性将在后续版本继续强化。
- Memo 手动拖拽持久化排序尚未包含在此版本。
