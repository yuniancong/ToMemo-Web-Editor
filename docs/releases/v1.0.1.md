# ToMemo Web Editor v1.0.1

面向 AI 批量导入顺序一致性的维护版本。

## 修复内容

ToMemo 手机端会依据记录时间显示 Memo。`v1.0.0` 在快速导入大量 AI 内容时，新记录的时间非常接近，手机端可能无法稳定复现网页预览顺序。

本版本为 AI 内容包中的新 Memo 生成确定性的阶梯时间：

- 页面第 1 条获得最新时间。
- 后续内容依次早 1 秒。
- ToMemo 使用最新优先排序时，手机端顺序与网页预览一致。
- 已有 Memo 的时间和内容不受影响。

## 用户影响

- 新导入的 AI 内容包拥有可预测的手机端顺序。
- `v1.0.0` 已生成的配置不会自动重新排序；需要使用本版本重新导入内容包。

## 兼容性

- 配置 schema 保持 ToMemo `1.0`。
- AI 内容包格式没有变化。
- 可直接从 `v1.0.0` 升级。

## 验证

- 自动化测试：18 项全部通过。
- TypeScript 与 Vite 生产构建通过。
- 阶梯时间生成和顺序保持具有独立回归测试。
